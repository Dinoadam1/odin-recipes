# odin-recipes

This project is to demonstrate a basic understanding of HTML. The goal of the project is to have mastery over setting up an HTML-boilerplate and setting up links and most likely lists with a fundamental understanding rather than superficial.

I personally hope to commit most of the things learned in this project to memory as this fundamental understanding and becoming more familiar with things that are going to become commonplace in a coding environment are the things I currently value the most, and are what I believe this course strives to teach.
